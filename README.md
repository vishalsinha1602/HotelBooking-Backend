# 🏨 HotelBooking-Backend

> A **production-ready Spring Boot backend** for a Hotel Booking system built with clean architecture, scalable design, and real-world backend practices.

---

## 🚀 Tech Stack

* ⚙️ Spring Boot
* 🌐 Spring MVC (REST APIs)
* 🗄️ Spring Data JPA (Hibernate)
* 🔐 Spring Security
* 🧪 JUnit & Testing
* 🐳 Docker *(Upcoming)*
* ☁️ CI/CD *(Upcoming)*

---

## 📌 Project Overview

**HotelBooking-Backend** is a backend system designed to simulate a **real-world hotel booking platform**.

It focuses on:

* 👤 User Management
* 🏨 Hotel & Room Management
* 📅 Booking System
* 🔐 Secure APIs
* 🧱 Scalable Architecture

---

## ⚙️ How to Run

```bash

# Navigate to project
cd HotelBooking-Backend

# Run application
./mvnw spring-boot:run
```
